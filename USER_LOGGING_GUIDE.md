## ✨ Що додано:

### 🔍 Автоматичне логування:
- **При кожному /start** зберігає user ID в список
- **Відрізняє нових та повертаючихся** користувачів  
- **Збирає статистику** по днях, тижнях, місяцях
- **Зберігає деталі:** ім'я, username, час першого/останнього входу

## 🔧 Налаштування адміністраторів:

### В файлі `bot/telegram_bot.py` знайди рядок:
```python
admin_users = [5324078617]  # Add Irina's Telegram user ID here
```

### Щоб знайти свій user ID:
1. Напиши боту [@userinfobot](https://t.me/userinfobot)
2. Він покаже твій ID: `Your user ID: 123456789`
3. Додай цей ID до списку admin_users

## 📊 Що можна відстежувати:

### ✅ Базова аналітика:
- **Загальна кількість** унікальних користувачів
- **Щоденна активність** - нові та активні користувачі
- **Тижнева/місячна статистика** - тренди росту
- **Найактивніші користувачі** - топ по використанню

### ✅ Детальна інформація:
- **Перший та останній вхід** кожного користувача
- **Кількість використань** /start кожним
- **Імена та username** для ідентифікації
- **Щоденний розподіл** активності

### ✅ Тренди:
- Скільки **нових користувачів** щодня
- Скільки **повертається** старих користувачів
- **Піки активності** по днях тижня
- **Ретеншен** - хто повертається

## 🚀 Як користуватись:

### Щоденний моніторинг:
```bash
/users  # Подивитись статистику
```

### Еженедельний аналіз:
- Скільки нових користувачів за тиждень?
- Який відсоток повертається?
- Коли найбільша активність?

### Ідеї для розвитку:
- Якщо багато нових - подумати про онбординг
- Якщо мало повертаються - поліпшити UX
- Якщо піки в певні дні - планувати оновлення

## ⚠️ Приватність:

- **Зберігаються тільки публічні дані** (ID, username, ім'я)
- **Не зберігається контент** повідомлень
- **Дані використовуються тільки для аналітики**
- **Доступ тільки адміністраторам**

## 🔄 Автоматичне логування:

Система працює **повністю автоматично**:
- При кожному `/start` додає користувача
- Оновлює статистику в реальному часі  
- Не потребує втручання
- Зберігає дані локально у файл
