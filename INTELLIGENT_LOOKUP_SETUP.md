# 🧠 Intelligent Lookup Setup

Система автоматического поиска информации о БАДах позволяет боту самостоятельно находить научно обоснованную информацию о новых добавках.

## 🚀 Как включить

### 1. Получить API ключ Brave Search (БЕСПЛАТНО)

1. Перейди на: https://brave.com/search/api/
2. Нажми "Get Started"  
3. Создай аккаунт или войди
4. Получи API ключ (бесплатно 1000 запросов/месяц)

### 2. Добавить ключ в .env

Открой файл `.env` и добавь:
```
BRAVE_SEARCH_API_KEY=your_api_key_here
```

### 3. Перезапустить бота

```bash
python bot/telegram_bot.py
```

## 🔍 Как это работает

### Без API ключа (текущее состояние)
```
Пользователь: "Родиола розовая"
Бот: ❌ Не знайшов автоматично інформацію. Підкажи, як його приймати?
```

### С API ключом (умный поиск)
```
Пользователь: "Родиола розовая" 
Бот: 🔍 Researching Родиола розовая...
     ✅ Found: Rhodiola Rosea
     🍽️ With food: Yes  
     ⏰ Best time: morning
     📝 Notes: Адаптоген, может повышать энергию (Найдено автоматически, уверенность: 0.8)
```

## 🎯 Источники информации

Система ищет информацию в порядке приоритета:
1. **Examine.com** (самый надежный)
2. **NIH Office of Dietary Supplements** 
3. **PubMed**
4. **Mayo Clinic**
5. **WebMD** 
6. **Healthline**

## 📊 Статистика

Система автоматически собирает статистику:
- Сколько БАДов исследовано
- Уровень уверенности в данных  
- Покрытие базы данных
- Дата последнего обновления

## 🔄 Как накапливаются знания

1. **Автоматический поиск** - бот ищет в интернете
2. **Пользовательский ввод** - когда пользователь добавляет вручную
3. **Кэширование** - результаты сохраняются для будущих пользователей
4. **Улучшение со временем** - база данных растет и улучшается

## 🛡️ Безопасность

- **Проверенные источники** - только надежные медицинские сайты
- **Уровень уверенности** - система показывает насколько уверена в данных
- **Безопасные значения по умолчанию** - если не уверена, рекомендует принимать с едой
- **Пользовательская проверка** - человек всегда может исправить

## 💡 Преимущества

### ✅ Для пользователей
- Мгновенная информация о любом БАДе
- Научно обоснованные рекомендации  
- Не нужно искать самостоятельно

### ✅ Для системы
- База знаний растет автоматически
- Улучшение качества со временем
- Меньше ручной работы по добавлению БАДов

## 🔧 Настройка уверенности

В коде можно изменить пороги:
- `confidence > 0.7` - высокая уверенность (зеленый)
- `confidence > 0.4` - средняя уверенность (желтый)  
- `confidence <= 0.4` - низкая уверенность (красный)
